# 🐓 GalloManager — Granja Pro

Sistema de gestión para granja de gallos miniatura y gallinas ornamentales.

## Stack
- **Frontend**: React 18 + Vite
- **Base de datos**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Deploy**: Vercel

## Variables de entorno (Vercel)

```
VITE_SUPABASE_URL=https://vgrevgmaybsacpcdkftk.supabase.co
VITE_SUPABASE_ANON_KEY=eyJ...
```

## Módulos
- Dashboard con estadísticas en tiempo real
- Inventario de aves con árbol genealógico
- Control de jaulas y ubicaciones
- Reproducción e incubación con barra de progreso
- Historial de salud y tratamientos
- Inventario de medicamentos con alertas de stock
- Registro de ventas e ingresos
